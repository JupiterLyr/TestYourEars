# Test Your Ears -- Waveform Generator

- Set a specific frequency and waveform to display the pitch while playing the sound. This can be used for hearing tests or to check the tuning of musical instruments.
- Easy to use and quick to show!

**Note**:
- To protect your hearing, please lower the volume before use.
- It is recommended to first conduct a trial with an amplitude of 0.5x.
